# Card Grid

Ein einfaches Kartenraster-Projekt mit Vite, SCSS und responsivem Layout.

## Projektübersicht

Dieses Projekt zeigt ein responsives Karten-Grid mit SCSS-basierten Design-Tokens, flexibler Gitterstruktur und Komponentenstil.

## Installation

1. `npm install`
2. `npm run dev`

## Ordnerstruktur

- `index.html` – Einstiegspunkt
- `src/main.js` – Anwendungsladepunkt
- `src/assets/scss/` – SCSS-Stile und Komponenten
